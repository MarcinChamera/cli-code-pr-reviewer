A modern, high-performance Snake game built with Python and Pygame.

## Features
- **Modern Aesthetics**: Uses a sleek dark theme with emerald green snake and vibrant food.
- **Dynamic Difficulty**: Speed increases as you grow longer.
- **Score Tracking**: Real-time score display and high score record (per session).
- **Responsive Controls**: Smooth movement using arrow keys.

## Requirements
- Python 3.x
- Pygame

## How to Play
1. **Install Dependencies**:
   ```bash
   pip install pygame
   ```
2. **Run the Game**:
   ```bash
   python snake.py
   ```
3. **Controls**:
   - Use **Arrow Keys** to move the snake.
   - Press **C** to play again after a game over.
   - Press **Q** to quit.

Enjoy the game!
